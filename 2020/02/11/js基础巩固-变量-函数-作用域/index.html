Javascript基础知识

细节

  • 多行字符串可以用反引号来表示。

  • ES6语法里有快速代入变量的方法，通过反引号来实现。

字符串的操作

  • 字符串可以使用类似数组的方法进行索引，但是不能更改字符串内容(字符串本身就是不可变的)

  • 如果是严格模式，则会报错。

字符串函数

  • toUpperCase(); 转变为大写。
  • toLowerCase(); 转变为小写。
  • indexOf(); 搜索指定字符串出现的位置。
  • substring(); 返回指定索引区间的字子串。

数组

  1. 数组长度可以被赋值，但是赋值后数组长度会随之变化。
  2. Array也有indexOf方法，用法和字符串相同。
  3. slice()截取Array部分元素，返回一个新Array(索引会发生变化)。
  4. push和pop对数组尾部进行操作。
  5. unshift()=>推入 和 shift() => 弹出 是对数组头部操作。
  6. reverse() 数组反转。
  7. splice()：修改Array的“万能方法”，它可以从指定的索引开始删除若干元素，然后再从该位置添加若干元素
  • 特别要注意，arr.splice()会返回删除掉元素所组成的数组。

  1. concat();将数组连接在一起，并且可以接收任意个元素和Array，并且自动把Array拆开，然后全部添加到新的Array里。

  2. join();把数组每个元素顺序连接起来，连接符号由传入的参数而定

Map(ES6)

  • Map是一组键值对的结构，具有极快的查找速度。
  • Map采用一对一的形式存储数据，每个元素就是一个键值对。

Set(集合)

  • Set和Map类似，也是一组key的集合，但不存储value。由于key不能重复，所以，在Set中，没有重复的key。

  • 要创建一个Set，需要提供一个Array作为输入，或者直接创建一个空Set。

iterable类型

  • 遍历Array可以采用下标循环，遍历Map和Set就无法使用下标。为了统一集合类型，ES6标准引入了新的iterable类型，Array、Map和Set都属于iterable类型。

  • 具有iterable类型的集合可以通过新的for … of循环来遍历。

函数

  • js中实际运用函数传入参数个数可以不和定义的参数个数相同。多的参数被舍弃，缺少的参数用undefined代替。

    函数自带索引arguments关键字

  • arguments指向一个类数组，其中是对传入参数的封装。

  • 可以通过Array.from()方法来实现对类数组的转变。ES6

rest参数(ES6)

  • 由于arguments参数不方便处理多出的参数，于是可以使用rest参数。

  • rest参数可以将多出的参数封装成一个数组。并且由rest存储。

  • 还有在js中，没事不要换行，因为js引擎会自动在每一行末尾加上一个分号。
