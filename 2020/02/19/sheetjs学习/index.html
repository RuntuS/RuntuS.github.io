  • XLSX.utils.sheet_to_json generates an array of objects
  • XLSX.utils.sheet_to_formulae generates a list of formulae

  • 输出关键字中属性解释

  1. v : raw value -> 原始数值。
  2. t : type (类型) -> 包括 b：Boolean，e：Error，n：Number，d：Date，s：Text，Z：Stub。
  3. w : formatted text (if applicable) -> 格式化文本
  4. r : rich text enchding -> 富文本编码
  5. h ： HTML rendering of the rich text (if applicable) : HTML格式呈现的富文本。
  6. [“!ref”]属性: 指定表的起始位置，在上述例子中,为A1:D2。(如果提前指定，则超过该属性的不会受理)
  7. [“!margins”]属性 : 代表页边距的对象。
