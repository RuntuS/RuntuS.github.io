定位(position),重点!!!!

  • 这里介绍五种常用值

  1. static(静态)

  2. relative

  3. fixed(固定)

  4. absolute(绝对)

  5. sticky(粘贴)

static

  • 首先,static就是元素默认定位,随文本流定位(即就在其原本位置).
    及时下面有top,bottom属性,也不会改变.

relative

  • relative定位: 该定位是相对于正常位置定位,并且不会脱离出文本流.
    原本位置不会被其他元素填充,如下图.

  • 设置前

  • 设置后

  • top:200px; 意思是距离原本的位置的top的200px的位置,所以向下平移动,同理若写为 left:200px; 是距离
    原本left的200px的位置,就会向右移动.
    ** 千万别和其他的搞混淆了. **

fixed

  • fixed是固定了元素相对于浏览器的位置,即无论怎么滑动,该元素都会跟着滑动(即相对浏览器位置不变),有点类似
    导航栏.由于会跟着滑动,所以该元素会脱离文本流,不占据空间.(即会覆盖其他元素.)

  • 效果图如下

  • 可以发现图片固定在浏览器的某个位置,并且脱离文本流,覆盖其他元素.

  • 这里补充一个用法,有时候想要把图片放在文字的下面,即不影响阅读,则可以用到 z-index 属性.(把其想象成z坐标轴)

  • 下方效果图可以看出,图片在文字的下方,达到我们想要的效果.

absolute

  • absolute的相对定位是相对于其最近的父元素,若无,则相对于整个.并且不会脱离出文本流,不占据空间.
    当其相对于整个HTML的时候,就可以等效于fixed的效果,只不过不随浏览器滑动.
    例如下.

sticky

  • sticky相当于relative与fixed的相互转换,其依赖于用户的滑动,当滑动一定程度之前,元素处于relative状态
    一旦超过了这个阀值,就会改变为fixed.
    重点谈谈这个阀值,这里谈论的上下滑动的情况,所以说阀值取决于top.
    当top设定这一个确定的px的时候,假设其为 Mpx,则当浏览器上方与该元素的top距离>=Mpx的时候.
    该元素处于relative状态,即不会随浏览器滑动.
    且此时的top:Mpx; 意思于relative里的意思相同.

    向下滑动的时候,若距离<=Mpx,则元素处于fixed状态,此时 top:Mpx 的意思与fix中的相同.

  • 这就是两种状态的相互转换.下图为超过阀值时的情况
