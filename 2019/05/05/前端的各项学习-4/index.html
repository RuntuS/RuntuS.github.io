前端的各项学习(4)

#从padding到margin,从内到外依次分析.

  • 下面的图可以大致展示其分布.

Padding

  • padding属性定义元素内容与边框距离(内边距).
    通常padding的值可以是具体的某个值(em,px)
    也可以是百分比(100%),该百分比是相对父元素而言.

  • padding的四种写法.

border 属性

  • 其简写属性为 border:width style color;
    其中style可以写之transparent 即透明边框.
    下面举几个常用的style.
    dotted:点线框.
    solid:实线框.

outline属性(外轮廓)

  • outline是紧贴着border外面的一层,主要内容是为了突出
    正文的内容.


  • 在白色border外面的点便是outline的展示.
    outline也是三个属性

margin 外边距

  • margin很单纯,它没有颜色,一般都是透明的,作用是清除周围的元素区域(把空间腾出来).
    一般可能用到的值为 auto 或者length -> px,em之类的.
