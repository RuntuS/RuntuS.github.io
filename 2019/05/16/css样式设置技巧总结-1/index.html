父子盒子设定

  • 其实是一个比较方便的设定,当两个及以上的盒子相互嵌套的时候,并且随时可能会改变
    这个整体的位置(但是不会改变内容)的时候,可以把外层父盒子的positon设置为relative,
    而内层的子元素position设置为absolute,那么在移动父盒子的时候,子盒子会跟着移动.
    特别强调一点,absolute元素的父元素如果没有设置position属性,那么将相对最外层的浏览器定位.(即body属性)
