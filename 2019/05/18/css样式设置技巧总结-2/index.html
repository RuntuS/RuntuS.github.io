css样式设置技巧总结(2)

关于旋转图片的样式设置(正面和背面图片不同)

  • 这个要用到关于css3的新样式,transform和transition来实现
    简单的动态效果.
    也要用到3d变换的样式 transform-style: preserve-3d;
    用3d变化要特别注意,一定要使用perspective(透视).这样才能展示3d效果.
    效果图如下



  • 只要把鼠标移到图片上时，就可以实现图片的旋转变换.
    下面简单谈谈步骤.

  1. 旋转盒子外部需要一个父盒子,父盒子的外部还再需要一个类似于舞台(stage)的父盒子,即要三层嵌套.

  2. 对于perstective设置在stage盒子里,它的原理相当于人距离舞台的距离,举个例子,假设图片宽度是300px.
    而人距离stage为500px(即perspective为500px),那么当图片沿着y轴旋转(竖直方向)的时候,当旋转90度时,
    图片的边离人就有200px远(最近距离),就像上图显示那样,那么就实现了3d的变化.

  3. transform的样式要设置在第二层父元素里,作用是规定样式为3d变换.同时需要设置变换时间,即transition(过渡)属性。

  4. 将两个图片重合,利用浮动定位来实现(具体步骤下列代码展示). 以及hover伪类的设定(毕竟是鼠标的动态效果).
