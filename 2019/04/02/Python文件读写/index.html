Python文件读写
  • 首先要定义一个文件变量,即要先进入文件里.有一种类似C语言进入文件的方法,
    即利用open来打开,close来关闭,但是这样可能会存在忘记close的情况,于是
    本章采用自动开关文件的方法处理.
    利用 with open(‘ ‘,’ ‘) as var: 来处理.
  • 下面介绍几个方法

  1. read()方法,读取文件的内容,并把内容返还给变量.
    write()方法,向文件写内容,并储存在文件中.

  2. rstrip()与strip()方法:前者是消去字符串末尾的换行,而后者是消去一切空白符.
    在读取文本内容时,如果想把多行都整合成一行,则用前者即可.

  3. readlines()与 文本for循环:
    两者都可以进行文本内容的行操作.

  • 特别提醒:无论是读文本,还是写文本,最终的结果都是以字符串的形式呈现的,意为,如果需要字符串以内的内容
    (比如数字),则需要强制转换来进行操作.
