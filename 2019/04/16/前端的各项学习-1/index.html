表单

  • 表单:表单是一个包含表单元素的区域.表单使用表单标签 <form> </form> 表单元素就放在其间
    下面说明下一个常用的表单元素和其对应的属性:
  • 表单标签的输入元素 <input>注意的是
    表单元素input的属性一般常用的有4种:文本域, 下拉列表 ,单选框(只能多选) ，复选框(可以多选).
    1）文本域 <input type="text"> 用户可以在其输入字符串.
    2）密码字段 <input type="password"> 即输入密码，可以保护不被看见，类似登陆账号时候输入密码.
    3）单选按钮 同理type=”radio”不过记得在input后记得再加一个 value ，否则按钮没有说明.
    特别注意,单选按钮几个value值必须是一样的,起一个标记的作用,否则不会起着单选的作用.
    4）复选按钮 type=”checkbox”,与3类似.

##额外补充:

  • 补充一个很妙的用法:
  1. 利用iframe元素，生成一个框架，该框架里面的内容也许不重要，但是一定为该元素命令一个名字 例如 name=”有点骚”。
  2. 那么在下面里使用 a 元素链接一个外部网站的时候，在a的target属性里设置 target=”有点骚“
  3. 那么在打开该网页的时候，新网页的展示将在框架里展示.
