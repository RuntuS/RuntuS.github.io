组件传值之子传父=>重点是:自定义事件

  • 事件一般分为浏览器自带事件和自定义事件，浏览器自定义事件例如click(点击事件),mouseon(鼠标在上事件)
  • 往往事件都会和v-on响应属性一起来用。
  • 子组件传值给父组件，需要通过事件响应来传值，同时需要$emit来辅助。
  • 大概流程是: 子组件发生事件响应，并且通过$emit传递想要的值。由于冒泡，父组件也会响应。
